# 隐私政策

## 数据收集
本插件**不会收集或存储任何个人用户数据**。  
所有输入内容仅用于生成嵌入向量的处理目的。

## 外部 API 使用
本插件会根据用户的配置调用外部的**嵌入模型 API**（例如 OpenAI、Azure 等）。  
请参阅相应服务提供商的隐私政策：
- OpenAI: https://openai.com/policies/privacy-policy
*（请根据您实际支持的 API 提供商进行添加或删除）*

## 数据保留
用户数据不会存储在插件端。  
所有请求和响应均保留在 Dify 工作流和外部 API 提供商内。

## 用户权利
用户可随时根据 Dify 的数据管理政策，在 Dify 内删除自己的数据。  
如有任何隐私相关问题，请联系：[Text to Embedding](https://github.com/lemonit-eric-mao/dify-plugins/tree/main/lemonit-eric-mao/Embedding)。
