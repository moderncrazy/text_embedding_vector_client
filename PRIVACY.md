# Privacy Policy

## Data Collection
This plugin **does not collect or store any personal user data**.
All inputs are processed only for the purpose of generating embeddings.

## External API Usage
The plugin calls external **Embedding Model APIs** (such as OpenAI, Azure, etc.) 
based on the user’s configuration.
Please refer to the respective provider’s privacy policy:
- OpenAI: https://openai.com/policies/privacy-policy
*(Add or remove providers depending on your supported APIs)*

## Data Retention
No user data is stored on the plugin side.  
All requests and responses remain within the Dify workflow and external API provider.

## User Rights
Users can delete their own data within Dify at any time according to Dify’s data management policy.  
For any privacy concerns, please contact: [Text to Embedding](https://github.com/lemonit-eric-mao/dify-plugins/tree/main/lemonit-eric-mao/Embedding).
